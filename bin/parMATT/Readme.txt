parMATT: Parallel multiple alignment with translations and twists for distributed-memory systems
Software version: 1.00
Version release date: February 20, 2018

The parMATT is licensed under the GNU public license version 2.0. 

If you find parMATT or its results useful please cite our work:
Shegay M., Suplatov D., Popova N., Å vedas V., Voevodin Vl. (2018) parMATT: Parallel multiple 
alignment with translations and twists for distributed-memory systems, submitted

The parMATT is based on the MATT algorithm and source code:
Menke M., Berger B., and Cowen L. (2008). Matt: local flexibility aids protein multiple 
structure alignment. PLoS computational biology, 4(1), e10.

You can use the following links to download useful content from the parMATT's web-page:

- Download the parMATT's User's Manual: https://biokinet.belozersky.msu.ru/sites/default/data/parMATT_user_manual.pdf
- Download small test set to validate your parMATT build: https://biokinet.belozersky.msu.ru/sites/default/data/Example_MAPK.tar.gz
- Download large test set to evaluate scalability of parMATT on your multiprocessor system: https://biokinet.belozersky.msu.ru/sites/default/data/Example_3.30.390.10.tar.gz

Please visit the parMATT's web-page for the latest version of software, documentation, and examples:
